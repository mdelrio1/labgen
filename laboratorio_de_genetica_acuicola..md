## Tablas

Este código 
```
Nombre columna 1| Nombre columna 2| Nombre *columna* 3| 
----------------|-----------------|-------------------|
valor 1|valor 2|valor 3|
```
generará la tabla siguiente:

Nombre columna 1| Nombre columna 2| Nombre *columna* 3| 
----------------|-----------------|-------------------|
valor 1|valor 2|valor 3|

#### Nota: se debe tener cuidado de tener un espacio antes y otro después de la tabla.
